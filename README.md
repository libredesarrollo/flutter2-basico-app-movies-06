# Películas App - Flutter

¡Bienvenido al proyecto de la aplicación de Películas! Esta es una aplicación desarrollada con Flutter que permite visualizar un listado de películas, obteniendo datos de servicios externos y mostrando detalles como la calificación de las mismas.

Este proyecto forma parte del aprendizaje práctico del ecosistema Flutter.

## 📚 Recurso de Aprendizaje

Este proyecto ha sido desarrollado siguiendo las pautas del libro:
**[Primeros pasos con Flutter](https://www.desarrollolibre.net/libros/primeros-pasos-flutter)**

Te recomiendo encarecidamente este recurso si quieres profundizar en tus conocimientos sobre Flutter.

---

## 🚀 Características

- **Listado de Películas**: Visualización dinámica de películas populares o recientes.
- **Calificaciones**: Integración de `flutter_rating_bar` para mostrar la valoración de cada film.
- **Consumo de API**: Uso del paquete `http` para la obtención de datos en tiempo real.
- **Interfaz Moderna**: Diseño limpio y funcional siguiendo los estándares de Material Design.

## 🛠️ Tecnologías Utilizadas

- [Flutter](https://flutter.dev/) - Framework de UI.
- [Dart](https://dart.dev/) - Lenguaje de programación.
- [http](https://pub.dev/packages/http) - Para realizar peticiones HTTP.
- [flutter_rating_bar](https://pub.dev/packages/flutter_rating_bar) - Para mostrar barras de estrellas.

## 🏁 Primeros Pasos

Para ejecutar este proyecto localmente, sigue estos pasos:

1. **Clonar el repositorio**:
   ```bash
   git clone <url-del-repositorio>
   ```

2. **Instalar dependencias**:
   ```bash
   flutter pub get
   ```

3. **Ejecutar la aplicación**:
   ```bash
   flutter run
   ```
